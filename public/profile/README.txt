HOW TO ADD YOUR PHOTOS
======================

Two photos are used on the site, both with background removed:

  1. hero.png         -> serious / arms-crossed shot (Home / Hero section)
  2. why-hire-me.png  -> pointing / animated shot   (Why Hire Me section)

STEP 1 — Remove the background
------------------------------
Easiest free option:
  • Open https://www.remove.bg in a browser
  • Upload the photo
  • Click "Download" — it gives you a transparent PNG

Or on macOS Preview:
  • Open the photo in Preview
  • Use the Markup toolbar -> "Instant Alpha" tool
  • Click+drag on the background to select it
  • Press Delete -> Save as PNG

STEP 2 — Save the files here
----------------------------
Save the two background-removed PNGs into THIS folder with these EXACT names:

  /Users/shariffomega/Desktop/shariff-portfolio/public/profile/hero.png
  /Users/shariffomega/Desktop/shariff-portfolio/public/profile/why-hire-me.png

STEP 3 — Refresh the browser
----------------------------
The site auto-detects both files and switches off the placeholder.
No code change needed.

Tip: keep each image around 1000-1500px wide for sharp rendering
without blowing up the page weight.
